Sobre o Projeto

Nosso projeto traz uma plataforma interativa que permite diversas análises dos dados do Exame Nacional do Ensino Médio de 2021, de modo a contribuir com informações que possam apontar para melhorias das políticas públicas para a educação nacional.

Durante o período em que estivemos em contato próximo com os dados analisados, constatamos que é alarmante o impacto dos diversos indicadores regionais, sociais e econômicos no desempenho dos candidatos ao ENEM. As diferenças que existem no contexto de um país de dimensões continentais como o Brasil, com todos os seus contrastes e desigualdades, se refletem nas informações que agora podem ser extraídas a partir das consultas e cruzamento dos dados que a plataforma permite, possibilitando ainda que cada interessado também faça suas próprias observações.

Para desenvolvimento e gestão do projeto foram utilizadas metodologias ágeis (Scrum e Kanban). O versionamento de código foi feito com o sistema Git, sendo que a documentação e hospedagem foram feitas na plataforma Github. A leitura, tratamento e análise inicial dos dados do ENEM 2021 foi feita com o Jupyter. A integração e população da base de dados resultante foi feita em Linguagem Python através da IDE Visual Studio Code. As consultas foram realizadas em linguagem SQL e, por fim, as informações resultantes estão disponíveis em dashboards gerados pelo Google Data Studio e Highcharts conectados a uma base de dados permanente em nuvem.
